# Skypro Wallet

Учебный проект для отработки верстки и базовой логики на React.

## Что есть в проекте

В проекте реализованы:
- Главная страница с хедером и навигацией
- Страница «Мои расходы» с таблицей
- Страница «Мои расходы» 
- Модалка авторизации/регистрации
- UI-компоненты кнопки и инпута
- Базовая адаптивность под небольшие экраны

### Главные технологии

- React
- Vite
- styled-components
- CSS (глобальный reset)

## Структура проекта

Основные файлы:
- `App.jsx` — корневой компонент приложения
- `App.css` — глобальные стили
- `pages/MyExpenses.jsx` — layout: хедер, навигация, контейнер контента
- `components/ExpenseTable.jsx` — таблица расходов + модалки «Новый расход» и «Редактирование»
- `components/AuthModal.jsx` — модальное окно входа/регистрации
- `components/Calendar.jsx` — модальное окно "Календаря" 
- `components/ExpenseChartModal.jsx` — модальное окно "Графика"
- `ui/Button.jsx` — переиспользуемая кнопка
- `ui/Input.jsx` — переиспользуемый инпут

## Функциональность

В приложении реализовано:
- Отображение списка расходов в таблице
- Переключение состояний модалки «Новый расход» / «Редактирование»
- Переключение режимов модалки авторизации: «Вход» и «Регистрация»
- Базовые фильтры и сортировка (UI-часть)
- Календарь

### Что делает каждая часть

- **MyExpenses (Layout)** — отрисовывает шапку сайта и область контента
- **ExpenseTable** — выводит расходы, фильтры, сортировку и форму добавления/редактирования
- **AuthModal** — отвечает за отображение форм входа и регистрации в одном компоненте
- **Button / Input** — единые UI-компоненты для кнопок и полей ввода
- **ExpenseChartModal / Calendar** — отвечает за отображение календаря и графика на странице "Анализ расходов"

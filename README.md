## Настройка ESLint и Prettier (React + Vite)

В проекте настроены:
- React + Vite
- ESLint 9 (Flat Config)
- Prettier 3
- Связка ESLint + Prettier без конфликтов (через `eslint-config-prettier`)
- Базовые скрипты для проверки и форматирования кода

### Что делает каждая часть
- **ESLint** — проверка качества кода (ошибки JS/JSX, правила хуков, неиспользуемые переменные). Форматированием не занимается.
- **Prettier** — единый стиль кода (кавычки, отступы, точки с запятой, переносы).
- **eslint-config-prettier** — отключает у ESLint правила, пересекающиеся с форматированием, чтобы исключить конфликты.
